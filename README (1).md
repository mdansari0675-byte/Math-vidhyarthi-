
# 📘 MathVidhyarthi

**MathVidhyarthi** is a free educational website for **Class 12 Mathematics (All Boards)**.  
It provides step-by-step solutions, MCQs, AI-based doubt solving, and customer support.

🌐 Live on **GitHub Pages**  
📚 Designed for **CBSE, Bihar Board, UP Board, ISC & State Boards**

---

## ✨ Features

- ✅ Class 12 Maths – All Boards
- ✅ Chapter-wise solved questions
- ✅ MCQ practice test
- ✅ AI Maths Chat (Frontend ready)
- ✅ Customer Support / Help Form
- ✅ Mobile friendly design
- ✅ Free hosting with GitHub Pages

## 🚀 How to Run Website (GitHub Pages)

1. Create a new repository on GitHub  
2. Upload `index.html` and `README.md`
3. Go to **Settings → Pages**
4. Select:
   - Branch: `main`
   - Folder: `/root`
5. Save and wait for 10–30 seconds

🔗 Your website will be live at:
```
https://your-username.github.io/mathvidhyarthi/
```
